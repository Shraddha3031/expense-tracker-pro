# Expense Tracker Pro

Expense Tracker Pro is a smart budget tracking web app built using Flask, HTML, CSS, JavaScript, and SQLite.

## Features
- User signup and login
- Add income and expense transactions
- Monthly budget setting
- Budget left calculation
- Smart spending suggestions
- AI spending prediction
- CSV, Excel, and PDF export
- Animated dashboard
- Dark mode toggle
- Mobile-friendly UI

## Tech Stack
- Python
- Flask
- HTML
- CSS
- JavaScript
- SQLite
- Chart.js
- ReportLab
- OpenPyXL
